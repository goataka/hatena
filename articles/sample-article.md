# はてなブログへようこそ

これはサンプル記事です。

## Markdownの使い方

このリポジトリでは、Markdownで記事を書くことができます。

### 特徴

- GitHub上で記事を管理
- 自動的にはてなブログにアップロード
- バージョン管理が簡単

### コード例

```javascript
console.log("Hello, Hatena Blog!");
```

## 記事の作成方法

1. `articles/` ディレクトリに新しい `.md` ファイルを作成
2. Frontmatterでタイトルやカテゴリーを設定
3. Markdownで本文を書く
4. Gitでコミット＆プッシュ
5. GitHub Actionsが自動的にはてなブログにアップロード

以上です！
